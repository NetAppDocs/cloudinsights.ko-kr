мЈјм„қ к·ңм№ҷ мһ‘м„ұ

02/25/2022 кё°м—¬мһҗ

мЈјм„қ к·ңм№ҷмқ„ мӮ¬мҡ©н•ҳм—¬ м •мқҳн•ң кё°мӨҖм—җ л”°лқј м—¬лҹ¬ мһҗмӮ°м—җ мЈјм„қмқ„ мһҗлҸҷмңјлЎң м Ғмҡ©н•  мҲҳ мһҲмҠөлӢҲлӢӨ. Cloud InsightsлҠ” мқҙлҹ¬н•ң к·ңм№ҷм—җ л”°лқј мһҗмӮ°м—җ мЈјм„қмқ„ н• лӢ№н•©лӢҲлӢӨ. Cloud Insightм—җм„ң мЈјм„қ к·ңм№ҷмқ„ нҸүк°Җн•  л•Ң к°ңлі„ мһҗмӮ° нҺҳмқҙм§Җм—җ мҲҳлҸҷмңјлЎң м„Өм •лҗң мЈјм„қмқҙ к·ңм№ҷ кё°л°ҳ мЈјм„қліҙлӢӨ мҡ°м„ н•©лӢҲлӢӨ.

мЈјм„қ к·ңм№ҷм—җ лҢҖн•ң мҝјлҰ¬лҘј л§Ңл“Өм–ҙм•ј н•©лӢҲлӢӨ.

  1. Cloud Insights л©”лүҙм—җм„ң * кҙҖлҰ¬ > мЈјм„қ к·ңм№ҷ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

  2. мғҲ мЈјм„қ к·ңм№ҷмқ„ м¶”к°Җн•ҳл Өл©ҙ * + к·ңм№ҷ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

    к·ңм№ҷ м¶”к°Җ лҢҖнҷ” мғҒмһҗк°Җ н‘ңмӢңлҗ©лӢҲлӢӨ.

  3. лӢӨмқҢмқ„ мҲҳн–үн•©лӢҲлӢӨ.

    1. мқҙлҰ„ * мғҒмһҗм—җ к·ңм№ҷмқ„ м„ӨлӘ…н•ҳлҠ” кі мң н•ң мқҙлҰ„мқ„ мһ…л Ҙн•©лӢҲлӢӨ.

      мЈјм„қ к·ңм№ҷ нҺҳмқҙм§Җм—җ мқҙлҰ„мқҙ лӮҳнғҖлӮ©лӢҲлӢӨ.

    2. Query * лҘј нҒҙлҰӯн•ҳкі  Cloud Insightsк°Җ мЈјм„қмқ„ м Ғмҡ©н•  мһҗмӮ°мқ„ мӢқлі„н•ҳкё° мң„н•ҙ мӮ¬мҡ©н•ҳлҠ” мҝјлҰ¬лҘј м„ нғқн•©лӢҲлӢӨ.

    3. Annotation(мЈјм„қ) * мқ„ нҒҙлҰӯн•ҳкі  м Ғмҡ©н•  мЈјм„қмқ„ м„ нғқн•©лӢҲлӢӨ.

    4. к°’ * мқ„ нҒҙлҰӯн•ҳкі  мЈјм„қ к°’мқ„ м„ нғқн•©лӢҲлӢӨ.

      мҳҲлҘј л“Өм–ҙ мЈјм„қмңјлЎң мғқмқј мқ„ м„ нғқн•ң кІҪмҡ° к°’мқҳ лӮ м§ңлҘј м§Җм •н•©лӢҲлӢӨ.

    5. м ҖмһҘ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ

    6. лӘЁл“  к·ңм№ҷмқ„ мҰүмӢң мӢӨн–үн•ҳл Өл©ҙ * лӘЁл“  к·ңм№ҷ мӢӨн–ү * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ. к·ёл Үм§Җ м•Ҡмңјл©ҙ к·ңм№ҷл“Өмқҙ м •кё°м ҒмңјлЎң мҳҲм•Ҫлҗң к°„кІ©мңјлЎң мӢӨн–үлҗ©лӢҲлӢӨ.

      лҢҖк·ңлӘЁ Cloud Insights нҷҳкІҪм—җм„ңлҠ” мЈјм„қ к·ңм№ҷмқ„ мӢӨн–үн•ҳлҠ” лҚ° мӢңк°„мқҙ мҳӨлһҳ кұёлҰҙ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҠ” мқёлҚұм„ңк°Җ лЁјм Җ мӢӨн–үлҗҳкі  к·ңм№ҷмқ„ мӢӨн–үн•ҳкё° м „м—җ мҷ„лЈҢлҗҳм–ҙм•ј н•ҳкё° л•Ңл¬ёмһ…лӢҲлӢӨ. мқёлҚұм„ңлҘј мӮ¬мҡ©н•ҳл©ҙ Cloud Insightsм—җм„ң лҚ°мқҙн„°мқҳ мғҲ к°ңмІҙ лҳҗлҠ” м—…лҚ°мқҙнҠёлҗң к°ңмІҙ л°Ҹ м№ҙмҡҙн„°лҘј кІҖмғүн•ҳкұ°лӮҳ н•„н„°л§Ғн•  мҲҳ мһҲмҠөлӢҲлӢӨ. к·ңм№ҷ м—”м§„мқҖ мқёлҚұм„ңк°Җ м—…лҚ°мқҙнҠёлҘј мҷ„лЈҢн•  л•Ңк№Ңм§Җ кё°лӢӨл ёлӢӨк°Җ к·ңм№ҷмқ„ м Ғмҡ©н•©лӢҲлӢӨ.

мЈјм„қ к·ңм№ҷ мҲҳм •

мЈјм„қ к·ңм№ҷмқ„ мҲҳм •н•ҳм—¬ к·ңм№ҷ мқҙлҰ„, мЈјм„қ, мЈјм„қ к°’ лҳҗлҠ” к·ңм№ҷкіј м—°кІ°лҗң мҝјлҰ¬лҘј ліҖкІҪн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

лӢЁкі„

  1. Cloud Insights л©”лүҙм—җм„ң * кҙҖлҰ¬ > мЈјм„қ к·ңм№ҷ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

    мЈјм„қ к·ңм№ҷ нҺҳмқҙм§Җм—җлҠ” кё°мЎҙ мЈјм„қ к·ңм№ҷ лӘ©лЎқмқҙ н‘ңмӢңлҗ©лӢҲлӢӨ.

  2. мҲҳм •н•  мЈјм„қ к·ңм№ҷмқ„ м°ҫмҠөлӢҲлӢӨ.

    н•„н„° мғҒмһҗм—җ к°’мқ„ мһ…л Ҙн•ҳм—¬ мЈјм„қ к·ңм№ҷмқ„ н•„н„°л§Ғн•ҳкұ°лӮҳ нҺҳмқҙм§Җ лІҲнҳёлҘј нҒҙлҰӯн•ҳм—¬ нҺҳмқҙм§Җлі„лЎң мЈјм„қ к·ңм№ҷмқ„ м°ҫм•„ліј мҲҳ мһҲмҠөлӢҲлӢӨ.

  3. мҲҳм •н•  к·ңм№ҷмқҳ л©”лүҙ м•„мқҙмҪҳмқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

  4. нҺём§‘ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ

    к·ңм№ҷ нҺём§‘ лҢҖнҷ” мғҒмһҗк°Җ н‘ңмӢңлҗ©лӢҲлӢӨ.

  5. мЈјм„қ к·ңм№ҷ мқҙлҰ„, мЈјм„қ, к°’ лҳҗлҠ” мҝјлҰ¬лҘј мҲҳм •н•©лӢҲлӢӨ.

к·ңм№ҷ мҲңм„ң ліҖкІҪ

мЈјм„қ к·ңм№ҷмқҖ к·ңм№ҷ лӘ©лЎқмқҳ л§Ё мң„м—җм„ң л§Ё м•„лһҳлЎң мІҳлҰ¬лҗ©лӢҲлӢӨ. к·ңм№ҷмқҙ мІҳлҰ¬лҗҳлҠ” мҲңм„ңлҘј ліҖкІҪн•ҳл Өл©ҙ лӢӨмқҢмқ„ мӢӨн–үн•©лӢҲлӢӨ.

  1. мқҙлҸҷн•  к·ңм№ҷмқҳ л©”лүҙ м•„мқҙмҪҳмқ„ нҒҙлҰӯн•©лӢҲлӢӨ.

  2. мӣҗн•ҳлҠ” мң„м№ҳм—җ к·ңм№ҷмқҙ лӮҳнғҖлӮ  л•Ңк№Ңм§Җ н•„мҡ”м—җ л”°лқј * мң„лЎң мқҙлҸҷ * лҳҗлҠ” * м•„лһҳлЎң мқҙлҸҷ * мқ„ нҒҙлҰӯн•©лӢҲлӢӨ.
